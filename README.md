# Lisy

Lisy is an educational multimodal application designed to make learning the Italian Sign Language (LIS) alphabet simple and interactive. The name combines "LIS" and "Easy", reflecting the project's goal of providing an accessible and engaging learning experience.

The application uses speech recognition and computer vision to guide users through the Italian sign language alphabet. Users can speak a word, which is automatically converted into a sequence of letters. For each letter, Lisy displays the corresponding LIS sign and asks the user to reproduce it. Through real-time hand tracking and gesture recognition, the system evaluates the performed sign and provides immediate feedback before moving to the next letter.

By combining voice input, visual guidance, and gesture recognition, Lisy offers an intuitive way to practice and learn the LIS alphabet while demonstrating the potential of multimodal human-computer interaction.
